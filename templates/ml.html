How The ML Works

Phase 5 — XGBoost Classifier

The ML model learns from completed trades to predict whether a new signal will be a win or loss.

Features used (18 total)

Price distance from EMA20/50/200 · RSI value and regime · ATR volatility ratio · Signal direction · HTF bias alignment · Order block width · SL distance · Recent BOS count · Liquidity events · FVG presence · Price momentum · Candle body strength

Training process

1. Run backtest → completed trades become training samples
2. Each trade's market context is converted to 18 numerical features
3. XGBoost trains a binary classifier (win=1, loss=0)
4. Cross-validation confirms the model generalises

Live scoring

Every new signal gets a ML probability score (0–100%) alongside the manual confidence score. High ML prob + high manual confidence = strongest signals.

Auto-improvement

Every trade that closes (win or loss) is automatically added to training data. The model gets smarter over time as it sees more real trade outcomes.
